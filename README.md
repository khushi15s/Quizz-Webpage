# 🎯 Quiz Game Web Page

A simple and interactive **Quiz Game Web Application** built using **HTML, CSS, and JavaScript**.
This project features a clean UI, lavender theme, and multiple-choice questions related to basic coding concepts.

---

## 🌟 Features

* 🎨 Attractive **light purple (lavender) UI**
* 💻 Designed for **laptop/desktop view**
* 📊 **Progress bar** to track quiz completion
* 🧠 15 **basic coding questions**
* ✅ Instant answer selection
* 🧮 Real-time **score tracking**
* 🔄 Restart quiz functionality

---

## 🚀 How to Run the Project

1. Download or copy all three files:

   * `index.html`
   * `style.css`
   * `script.js`

2. Make sure all files are in the **same folder**

3. Open `index.html` in your browser
